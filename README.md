# Индейка — семейная CRM

Мини-CRM на React + Vite для управления ценами, доставкой, остатками, акциями и генерацией прайсов в PNG.

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация на Vercel

1. Создайте новый репозиторий на GitHub и загрузите туда содержимое этой папки.
2. Откройте Vercel → Add New → Project.
3. Импортируйте репозиторий.
4. Vercel автоматически определит Vite. Нажмите Deploy.
5. После публикации вы получите постоянную ссылку.

Данные сохраняются в localStorage конкретного браузера. Для синхронизации между телефонами позже можно подключить Supabase или Firebase.
